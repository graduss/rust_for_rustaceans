# Rust for Rustaceans — конспект и практика

Личный учебный репозиторий по книге Джона Йенсо (Jon Gjengset) *Rust for Rustaceans* (русский перевод). Здесь собраны материалы глав, дополнительные примеры с разбором и код упражнений.

## Структура

```
book/        Тексты глав: Глава_N_<Тема>.md (+ PDF-версии)
examples/    Практика к главам: Глава_N_<Тема>_практика.md
             — «Примеры» (код с разбором) и «Задания» (упражнения)
src/         Код примеров и упражнений, по модулю на главу
```

## Код

Проект — обычный бинарный крейт (edition 2024, без зависимостей):

```sh
cargo run    # запустить подключённые примеры
cargo check  # быстрая проверка компиляции
```

- `src/main.rs` объявляет модули глав (`mod c2;` и т. д.) и вызывает функции-сниппеты.
- `src/cN.rs` — код главы N:
  - `fX_Y()` — «Пример X.Y» из файла практики;
  - `eX_Y()` — задание Y из раздела X («Задания»).

Примеры также можно проверять в [Rust Playground](https://play.rust-lang.org/).
